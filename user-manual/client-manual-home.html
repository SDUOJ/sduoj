# 主页

主页用轮播图展示最新事件。在轮播图下方有两个显示框，分别显示最近的公告以及搜索框。

另外，主页的导航栏由五部分组成，从左到右分别是「主页」、「题库」、「比赛」、「提交」、「个人」。可以点击具体的入口进入指定页面。
